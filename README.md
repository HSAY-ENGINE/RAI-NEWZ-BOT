# RAI NEWZ BOT 🤖

An AI-powered news bot built with Python that automatically
fetches the latest AI news and delivers it to Telegram twice daily.

## What It Does
- Monitors 7 YouTube channels for new AI videos
- Tracks 10 AI news websites (OpenAI, Anthropic, Google DeepMind, etc.)
- Summarizes every piece of content using Google Gemini AI
- Gives a WATCH or SKIP verdict for YouTube videos
- Sends everything to Telegram at 8AM and 8PM daily
- Never sends duplicate content

## Tech Stack
- Python
- Google Gemini AI API
- Telegram Bot API
- feedparser (RSS feeds)
- schedule (automation)
